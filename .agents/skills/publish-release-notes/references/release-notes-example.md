### What's New

- Tap a completed received file to open it in a compatible app.

### Improvements

- Improved compatibility when receiving files from newer CoLink devices that do not include a checksum.

### Fixes

- Fixed LAN file transfers being marked as failed after all file data had been sent successfully.
